## 🆘 Having Issues?

### Installation Problems
👉 See **INSTALLATION.md** → Troubleshooting section

### Port Already in Use
```bash
npm run dev -- --port 3000
```

### Changes Not Showing
- Hard refresh: `Ctrl + Shift + R` (Windows) or `Cmd + Shift + R` (Mac)
- Clear browser cache
- Restart dev server

### Node Version Too Old
Check version:
```bash
node --version
# Should be v18.0.0 or higher
```
Download latest: https://nodejs.org/

## 🌐 Deployment

After running `npm run build`, you'll have a `/dist` folder.

**Upload this folder to any hosting service:**
- Netlify (drag & drop)
- Vercel (connect Git repo)
- AWS S3 (static hosting)
- Any web server

**That's it!** No server-side code needed.
